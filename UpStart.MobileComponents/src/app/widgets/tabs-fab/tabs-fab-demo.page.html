 Implementation Details

1. Hidden Spacer Tab

A disabled, invisible tab button creates space for the FAB in the center:

<ion-tab-button class="fab-spacer" disabled="true" style="visibility: hidden;">

2. Centered FAB Positioning

CSS for perfect centering above the tab bar:

left: 50%; transform: translateX(-50%); bottom: 50px;

3. Icon Switching

Each tab contains both outline and solid icons. CSS toggles visibility:

.tab-selected .solid-icon { display: block; }
.tab-selected .outline-icon { display: none; }

4. iOS Safe Area

Handles iOS home indicator:

padding-bottom: calc(env(safe-area-inset-bottom) - 9px);
